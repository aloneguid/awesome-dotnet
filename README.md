# awesome-dotnet

Awesome .NET libraries - A community-curated list of awesome .NET libraries and tools. Making sure there is no single point of failure.

## How to Contribute

1. [Create a new issue](../../issues/new/choose) and fill in all the information.
2. If your suggestion gets at least **5 likes (üëç)**, or just owner's single like, it will be automatically added to this list! The list of issues is checked automatically every hour.
